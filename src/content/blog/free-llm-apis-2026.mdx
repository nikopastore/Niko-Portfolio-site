## API Compatibility

All providers listed are **OpenAI SDK-compatible**. You can use:

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.provider.com/v1",
    api_key="your-key"
)

response = client.chat.completions.create(
    model="model-name",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

Just change the `base_url` and `api_key` for each provider.
