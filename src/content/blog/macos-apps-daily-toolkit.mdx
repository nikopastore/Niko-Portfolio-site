**TL;DR**: Most "best macOS apps" lists are wishlists, not real toolkits. Here's what I actually have installed after years of testing — organized by what native macOS feature they fix or replace.

---

Every Mac user has been there: you read a "50 Best macOS Apps" article, install half of them, and three weeks later you've uninstalled everything except two.

I got tired of that cycle. So I started tracking which apps actually survived — the ones still running months later, not the ones that looked good in a screenshot.

Here's the result: my actual daily toolkit, organized by what problem each app solves.

---

## Fixing What Apple Broke

These apps fix things that should work better out of the box.

### Raycast → Replaces Spotlight
Spotlight is fine. Raycast is what Spotlight should have been. Snippets, window management, clipboard history, app launching — all keyboard-first, all fast. I open it 50+ times a day.

### AltTab → Fixes Window Switching
macOS Cmd+Tab switches between apps, not windows. AltTab shows every window. It's the behavior Windows has had for decades. Essential.

### MacMouseFix → Makes Mice Usable
If you use any non-Apple mouse, scrolling feels wrong on macOS. This fixes it. Should be built into System Preferences.

### Rectangle → Adds Window Snapping
Windows has had drag-to-snap since 2009. macOS still doesn't (Stage Manager doesn't count). Rectangle adds it. Free, fast, done.

## Replacing Weak Native Apps

Apple's built-in apps that need better alternatives.

![Clean Mac workspace setup](/blog/images/mac-workspace.jpg)
*A clean setup requires the right tools underneath.*

### IINA → Replaces QuickTime
QuickTime can't play half the video formats that exist. IINA handles everything, looks native, and doesn't ask you to convert anything.

### Pearcleaner → Replaces... Nothing (But Should Exist)
macOS has no real uninstaller. Dragging apps to trash leaves cruft everywhere. Pearcleaner actually removes everything — preferences, caches, login items.

### Keka → Replaces Archive Utility
The built-in archiver works for zip files. Keka handles everything else — 7z, RAR, tar.gz — without thinking about it.

## Power Tools Worth the Learning Curve

These take time to set up but pay dividends forever.

### BetterTouchTool
Customize every input device: trackpad gestures, mouse buttons, keyboard shortcuts, Touch Bar (RIP). The settings are overwhelming at first. Once configured, it's invisible and essential.

### Keyboard Maestro
If you do the same sequence of actions more than twice, Keyboard Maestro can automate it. Steeper learning curve than Raycast, but more powerful when you need it.

### Obsidian
Note-taking that scales. Plain markdown files, local storage, infinite customization. The learning curve is real — give it two weeks before judging.

---

## Small Tools, Big Impact

Tiny apps that solve one problem perfectly.

### CleanShot X
Screenshots that don't require three more apps to be useful. Annotate, blur, upload, share. Worth every penny. (Shottr is a solid free alternative.)

### Dropover
Drag something, drop it on the shelf, keep working, drag it to the real destination later. Sounds silly until you use it. Then you can't stop.

### Maccy
Clipboard history. Free, lightweight, keyboard-driven. Paste something from an hour ago instead of hunting for it.

### Ice
Menu bar getting crowded? Ice hides icons until you need them. Clean, native, simple.

### Amphetamine
Keep your Mac awake during a long download or presentation. One job, done well.

## The "Everyone Recommends But I Don't Use" List

Popular apps that didn't stick for me:

### Stats
System monitoring in the menu bar. Cool for a week, then you realize you never actually look at it. Uninstalled.

### Hidden Bar
Menu bar manager. Ice does the same thing better.

### AppCleaner
The classic uninstaller. Pearcleaner is more thorough and looks better.

### The Unarchiver
Works fine. Keka is just better.

## What I'm Still Evaluating

### Arc Browser
Either the future of browsing or a solution looking for a problem. I switch between loving it and going back to Safari. Jury's still out.

### Things 3 vs Reminders
Things is beautiful. But Apple's Reminders got good enough that I'm not sure it's worth $50 anymore. Using both, deciding.

### Linear
Issue tracking that doesn't make me want to quit my job. If you're stuck with Jira, Linear feels like therapy. But it's a team decision, not a personal one.

---

## The Installation Philosophy

My rule: **If I don't open it in two weeks, it's gone.**

Most apps fail this test. The ones above survived. They either solve a real problem or improve a workflow I use daily.

No app earns a permanent spot just by being well-designed. It has to be useful enough that I'd notice if it disappeared.

That's the bar.
