"OpenClaw is getting out of hand."

That was the title of a Reddit post that caught my attention. The author wasn't complaining — they were genuinely amazed that OpenClaw could now control their entire phone through the MobileRun skill.

And that's just one capability. After going through recent release notes, community setups, and my own daily usage, here's a breakdown of what OpenClaw can actually do right now.

## What OpenClaw Is (Quick Context)

OpenClaw is an AI agent platform that connects to your messaging apps (WhatsApp, Telegram, Discord, iMessage) and acts as an autonomous assistant. You message it, it does things — not just answers questions, but actually executes tasks.

The key difference from Claude or ChatGPT: OpenClaw has **tools**. It can read your files, send emails, manage calendars, control browsers, run shell commands, and connect to hundreds of services through MCP (Model Context Protocol) servers.

## The Core Capabilities

### 1. Messaging Integration (The Foundation)

OpenClaw connects to the apps you already use:
- **WhatsApp** — via the WA Web bridge
- **Telegram** — native bot integration
- **Discord** — full server/channel access
- **iMessage** — macOS only, via AppleScript
- **Signal** — self-hosted bridge
- **Slack** — via app integration
- **IRC** — for the old school

You message it like a friend. It responds like an assistant with superpowers.

### 2. Local File Access

Point OpenClaw at any folder and it can:
- Read and write files
- Search content semantically
- Process documents (PDFs, images, audio)
- Organize and rename files
- Track changes over time

The `MEMORY.md` pattern lets you give it persistent context about your projects, preferences, and ongoing work.

### 3. Browser Control

With the browser skill, OpenClaw can:
- Navigate websites
- Fill forms
- Extract data
- Take actions (book flights, submit applications)
- Handle multi-step workflows

This is where it gets "out of hand" — you can literally say "apply to this job" and it fills out the application.

### 4. Phone Control (MobileRun)

The newest capability: OpenClaw can control your phone through the MobileRun skill. This means:
- Opening apps
- Sending messages
- Navigating UI
- Taking screenshots
- Automating mobile workflows

No more "I can't do that, I'm just an AI." It actually does it.

### 5. Multimodal Memory Search

As of v2026.3.11, OpenClaw can index images and audio into searchable memory:

> "Drop screenshots of UI errors, flow diagrams, or design comps into a folder, let OpenClaw index them, and ask: 'What's wrong in this error?' or 'Find similar past UI issues.'"

Use recorded calls, standups, or training sessions as a searchable archive. This uses Gemini's embedding model under the hood.

### 6. Sub-Agent Spawning

OpenClaw can spawn child agents for complex tasks:
- Break down problems into sub-tasks
- Run multiple agents in parallel
- Resume parent conversations with summaries
- Handle multi-step debugging flows

The ACP (Agent Communication Protocol) sessions can now resume instead of always starting fresh.

### 7. Cron Jobs & Scheduled Tasks

Set up recurring automation:
- Daily standup summaries
- Inbox digests
- Monitoring alerts
- Report generation
- Data backups

Cron jobs are now cleanly isolated from ad-hoc agent sends, so schedules don't leak into random chats.

### 8. Local-First with Ollama

If you want to run everything locally:
- First-class Ollama setup from onboarding
- Curated model suggestions for different tasks
- Hybrid local + cloud mode
- Privacy-heavy, on-device workflows

No cloud keys required if you don't want them.

## Getting Started

If you want to try it:

1. Install: `npm install -g openclaw`
2. Run: `openclaw setup`
3. Connect a messaging app (Discord is easiest)
4. Start messaging

The [official docs](https://docs.openclaw.ai) walk through setup. The [Discord community](https://discord.com/invite/clawd) is where people share configs and troubleshoot.
